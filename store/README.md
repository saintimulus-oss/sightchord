# Store Metadata

This directory holds store-submission copy grouped by platform and locale.

## Structure

- `google-play/<locale>/`
- `app-store/<locale>/`
- `assets/`

Policy and submission drafting files also live alongside the metadata, including:

- `google-play/data-safety-notes.md`
- `google-play/app-content-notes.md`
- `app-store/app-privacy-notes.md`

## Current Locales

- `en-US`
- `ko-KR` for Google Play
- `ko` for App Store

Add more locales by copying an existing locale folder and translating the text files inside it.
