Chordest는 코드, 화성, 즉흥 연습을 한 곳에서 진행할 수 있도록 설계된 로컬 중심 음악 연습 앱입니다.

12개 모든 루트에서 진행을 생성하고, 자유 모드와 조성 기반 모드를 전환하며, Smart Generator로 가중치 기반 화성 진행을 탐색할 수 있습니다. 내장된 Chord Analyzer를 통해 조성 중심, 로마 숫자, 화성 기능, 요약 결과를 함께 확인할 수 있습니다.

주요 기능:
- 12개 루트 전체 코드 연습
- 다이어토닉 로마 숫자를 지원하는 Key 모드
- Smart Generator 기반 진행 생성
- 이론 중심 결과를 보여주는 Chord Analyzer
- 메트로놈, 자동 진행, BPM 제어, 표기 옵션
- 영어, 스페인어, 중국어 간체, 일본어, 한국어 UI 지원

현재 Chordest는 설정과 진행도를 기기에 저장하며, 계정 생성 없이 바로 사용할 수 있습니다.
