# Paywall 한국어 카피

기준일: 2026-03-25

## 카피 원칙

- 과장하지 않는다.
- 가격은 스토어 상품 정보에서만 보여준다.
- 구매하지 않아도 무료 기능 사용은 계속 가능하다고 분명히 말한다.
- `beta`, `coming soon`, `준비 중` 같은 표현은 쓰지 않는다.

- `Chordest Premium`

## 본문

- 고급 화성 옵션 진입 시:
  - `비다이아토닉 컬러 옵션과 고급 tension은 프리미엄 해금 대상입니다.`

## 상태 메시지

- 오프라인 캐시 안내
  - 제목: `최근 확인된 해금을 사용 중입니다`
  - 본문: `지금은 스토어 연결이 되지 않아, 최근 확인된 프리미엄 해금 캐시를 사용하고 있습니다.`
- 스토어 연결 불가
  - 제목: `스토어 연결 불가`
  - 본문: `지금은 스토어에 연결할 수 없습니다. 무료 기능은 그대로 사용할 수 있습니다.`
- 상품 정보 미조회
  - 제목: `상품 정보를 아직 불러오지 못했습니다`
  - 본문: `프리미엄 상품 정보가 없습니다. 잠시 후 다시 시도하거나 Play Console 설정을 확인해 주세요.`
- 구매 성공
  - 제목: `프리미엄이 해금되었습니다`
  - 본문: `이 기기에서 영구 프리미엄 해금이 활성화되었습니다.`
- 복원 성공
  - 제목: `구매를 복원했습니다`
  - 본문: `스토어에서 프리미엄 해금을 복원했습니다.`
- 복원 실패
  - 제목: `복원할 구매가 없습니다`
  - 본문: `이 스토어 계정에서 일치하는 프리미엄 해금을 찾지 못했습니다.`
- 취소
  - 제목: `구매를 취소했습니다`
  - 본문: `과금되지 않았습니다. 무료 기능은 계속 사용할 수 있습니다.`
- 보류
  - 제목: `구매 대기 중`
  - 본문: `스토어에서 구매를 대기 상태로 표시했습니다. 확인이 끝나면 프리미엄이 활성화됩니다.`
- 실패
  - 제목: `구매에 실패했습니다`
  - 본문: `구매를 완료하지 못했습니다. 잠시 후 다시 시도해 주세요.`
- 이미 보유
  - 제목: `이미 해금되었습니다`
  - 본문: `이 스토어 계정에는 이미 프리미엄 해금이 있습니다.`
