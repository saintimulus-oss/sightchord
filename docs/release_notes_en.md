기준일: 2026-03-25

## English Draft

### Summary

Chordest is now focused on a clear Android launch scope. The core chord generation and practice loop stays free, while Smart Generator and advanced harmonic controls are grouped under a one-time premium unlock.

### What’s new

- The app is now positioned around Chordest, with Analyzer kept as a secondary companion tool.
- Study Harmony remains sealed off from the release user path.
- A one-time premium unlock flow is in place for Smart Generator and advanced harmony controls.
- Startup stability and billing cache resilience were improved for release readiness.
- Key user-facing copy was cleaned up across the first-run flow and paywall.
- Android release readiness, QA, and Play Console handoff docs were finalized.

### Free features

- Core Chordest generator loop
- Basic chord display, inversions, and slash bass
- Metronome and custom metronome sound
- Setup assistant, language, theme, and standard settings
- Conservative Analyzer basics

### Premium unlock

- Smart Generator mode
- Secondary dominants, substitute dominants, and modal interchange
- Advanced tensions
- Advanced Smart Generator controls

### Not included in this release

- Study Harmony hub and session flows
- Analyzer variation generation
- Subscriptions
- Server-backed entitlement validation or sync
